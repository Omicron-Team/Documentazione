\subsection{A}
\TermineGlossario{API}
\DefinizioneGlossario{Acronym of Application Programming Interface. In computing it is an interface that defines interactions between multiple software applications or mixed hardware-software intermediaries. They allow products or services to communicate with other products or services without knowing how they are implemented.}
\\
\TermineGlossario{Amazon Web Services (AWS)}
\DefinizioneGlossario{US company owned by Amazon, which provides cloud computing services on an on-demand platform of the same name.}
\\
\TermineGlossario{Asincrony}
\DefinizioneGlossario{It allows a software to continue certain activities in parallel to the main program flow, so the caller does not have to actively wait for the computation of the invoked code to finish.}
