\subsection{N}
\TermineGlossario{Next.js}
\DefinizioneGlossario{Next.js is an open-source React front-end development web framework that enables functionality such as server-side rendering and generating static websites for React based web applications.}
\TermineGlossario{Node.js}
\DefinizioneGlossario{Node.js is an open-source, cross-platform, event-driven, back-end JavaScript runtime environment that executes JavaScript code outside a web browser. Node.js lets developers use JavaScript to write command line tools and for server-side scripting.}
