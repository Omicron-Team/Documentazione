\subsection{S}
\TermineGlossario{Server-side rendering}
\DefinizioneGlossario{Rendering process of HTML and CSS pages where the server renders the page on every client request.}
\\
\TermineGlossario{Serverless (framework)}
\DefinizioneGlossario{The Serverless Framework is a free and open-source web framework written using Node.js. Serverless is the first framework developed for building applications on AWS Lambda.}
\\
\TermineGlossario{Static generation}
\DefinizioneGlossario{Rendering process of HTML and CSS where the server renders pages before requests, for example at the time the build is run. Already rendered HTML pages are cached and ready to be sent at the time of requests.}
\\
\TermineGlossario{Stripe}
\DefinizioneGlossario{Company that primarily offers payment processing software and application programming interfaces (APIs) for e-commerce websites and mobile applications.}
\\
