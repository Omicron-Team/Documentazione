\section{N}

\TermineGlossario{Node.js}
\DefinizioneGlossario{Consiste in una runtime di JavaScript open source e multipiattaforma orientato agli eventi (rende possibile l’I/O asincrono) per l’esecuzione di codice JavaScript.}
\\
\TermineGlossario{NoSQL}
\DefinizioneGlossario{Movimento che promuove sistemi software dove la persistenza dei dati è in generale caratterizzata dal fatto di non utilizzare il modello relazionale.}
\\
\TermineGlossario{Next.js}
\DefinizioneGlossario{Framework web open-source di sviluppo front-end che abilita funzionalità come il rendering lato server e la generazione di siti web statici per applicazioni web basate su React.}
\\
\TermineGlossario{Numpy}
\DefinizioneGlossario{Libreria open source per il linguaggio di programmazione Python, che aggiunge supporto a grandi matrici e array multidimensionali insieme a una vasta collezione di funzioni matematiche di alto livello per poter operare efficientemente su queste strutture dati.}
\\
